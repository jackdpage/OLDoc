Pylux
=====

Pylux is a program for managing stage and entertainment lighting 
documentation. It allows you to create reports using Jinja and SVG diagrams of 
lighting plots.

Features
--------

- Create documentation in any form using Jinja
- Reference a library of fixtures and gels
- Produce SVG plans of plots

Installation
------------

Pylux can be installed from the PyPI:

    pip install pylux
